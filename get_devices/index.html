Persistent Bluetooth permission via getDevices API

This is a simple test to check if the bluetooth permission is persistent when using getDevices API.

Step 1

Press the button to load the code to the Espruino IDE. From there, flash any Bluetooth capable Espruino device.

View source.

Step 2

Before start pairing process, make sure you remove the Bluetooth device form the list of trusted devices of the page by clicking on the lock button next to the URL and clicking the "Reset permission" in the bluetooth section.
Once running, start pairing process.

Step 3

Press the button to refresh the page.

Step 4

Press the button after refresh.
