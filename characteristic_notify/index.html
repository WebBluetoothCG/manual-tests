Bluetooth Characteristic Read w/Notifications

This is a simple test to subscribe to characteristic value change notifications.

Step 1

Press the button to load the code to the Espruino IDE. From there flash any Bluetooth capable Espruino device.

View source.

Step 2

Once running, start the test.

Status: 


  

  
  

    Bluetooth is not present.
  

  

    Bluetooth requires a secure context (HTTPS). For development purposes it
    does support HTTP, but only the on the loopback adapter (i.e. "localhost").
  

  

    Bluetooth is supported, but unavailable. The radio may be disabled
    or blocked by your user agent.
